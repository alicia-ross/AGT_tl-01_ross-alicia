Animation

Exemple d'animation où une boîte se déplace horizontalement.


                .animated-box {
                    width: 50px;
                    height: 50px;
                    background-color: red;
                    animation: example-animation 2s infinite;
                }

                @keyframes example-animation {
                    0% { transform: translateX(0); }
                    50% { transform: translateX(100px); }
                    100% { transform: translateX(0); }
                }

B

Background-size

La propriété CSS background-size permet de spécifier la taille de l'image de fond dans un élément.


                .background-example {
                    background-image: url('exemple.jpg');
                    background-size: cover;
                }

Border-radius

La propriété CSS border-radius permet de spécifier des coins arrondis pour un élément. Vous pouvez utiliser une valeur spécifique en pixels ou en pourcentage pour déterminer le rayon de l'arrondi.


                .border-example {
                    border-radius: 50%;
                }

Box-shadow

La propriété CSS box-shadow permet d'ajouter une ombre à un élément. Vous pouvez spécifier l'ombre horizontale, verticale, le flou, la propagation et la couleur de l'ombre.


                .box-shadow-example {
                    box-shadow: 10px 10px 10px rgba(0, 0, 0, 0.5);
                }

Filter

La propriété CSS filter permet d'appliquer des effets visuels (comme le flou, la saturation des couleurs, etc.) aux éléments HTML.

Flou


                .example-image.blur {
                    filter: blur(5px);
                }

Saturate


                .example-image.saturate {
                    filter: saturate(200%);
                }

Invert


                .example-image.invert {
                    filter: invert(100%);
                }

T

Text-decoration

La propriété CSS text-decoration permet de définir la décoration du texte, telle que le soulignement, la ligne au-dessus ou la ligne à travers.

Ceci est un exemple de texte avec une décoration de soulignement.

                .example {
                    text-decoration: underline;
                }

Transform

La propriété CSS transform permet d'appliquer des transformations à un élément, comme la rotation, l'échelle, l'inclinaison et la translation.


                .triangle {
                    transform: rotate(180deg);
                }

                .losange {
                    transform: rotate(45deg);
                }

                .ovale {
                    transform: scaleY(2);
                }

                .rectangle {
                    transform: skewX(30deg);
                }

                .trapeze {
                    transform: perspective(500px) rotateX(30deg);
                }

                .octagon {
                    transform: rotate(90deg);
                }

Transition

La propriété CSS transition permet de spécifier une transition entre les états d'un élément lorsqu'un changement de style est détecté.


                .custom-button {
                    transition: background-color 0.3s ease;
                }
